# link-app
## Webapp para guardar links de paginas mas usadas

1-Cona o descarga el repositorio a tu pc

2-Ejecuta el comando $ npm i o npm install (dentro de la carpeta)

3-Inserta tu clave privada y demas informacion proporcionada por fiirebase

4-Ejecuta el comando $ npm run serve para arraancar el servidor de desarrollo

5-Ejecuta el comando $ npm run build para la compilacion y empaquetado de produccion
